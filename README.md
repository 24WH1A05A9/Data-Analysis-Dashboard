Restaurant Demand Forecasting & Food Waste Optimization (Power BI + SQL Project)
Project Objective

The objective of this project is to analyze restaurant data to improve demand forecasting, reduce food wastage, and enhance profitability. The system helps restaurant owners make data-driven decisions by understanding customer behavior, demand patterns, and operational inefficiencies.

Dataset Used
Orders Dataset
Items Dataset
Inventory Dataset
Waste Log Dataset
External Signals Dataset (Weather, Weekend, Festival)
Questions (KPIs)
Which items are most frequently ordered?
Which items contribute the most to food waste?
How does demand vary across weekdays and weekends?
What is the impact of festivals on customer demand?
How does weather (rain, temperature) affect order patterns?
Which items generate the highest profit?
What is the total revenue and profit per restaurant?
What is the waste percentage and waste cost?
What are the main reasons for food wastage?
Which items have high demand but also high waste?
Process
Collected and integrated multiple datasets (orders, items, waste, inventory, external factors)
Cleaned data by handling missing values and removing inconsistencies
Performed SQL joins to create a unified dataset
Created calculated fields such as profit, waste cost, and demand metrics
Conducted analysis using SQL queries (demand, waste, profit, trends)
Built interactive dashboards using Microsoft Power BI
Applied filters and slicers for dynamic data exploration
Dashboard
Interactive Power BI dashboard with:
Bar Charts (Top items, waste distribution)
Line Graphs (Demand & waste trends over time)
Pie Charts (Waste reasons, category distribution)
Filters (Date, item, restaurant)

Project Insights
A small number of items contribute to the majority of revenue
The same high-demand items also generate significant waste
Weekend and festival days show increased customer demand
Weather conditions (especially rain) influence ordering patterns
Expired and spoiled items are the major causes of food waste
High waste cost is linked to poor inventory and demand estimation
Profitability varies significantly across menu items
Final Conclusion

By using data analytics, restaurants can significantly reduce food waste and improve profitability. The insights from this project help in optimizing inventory management, adjusting food preparation quantities, and planning better during peak demand periods like weekends and festivals. Overall, this system supports smarter decision-making and promotes sustainable restaurant operations.
