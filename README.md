# Restaurant Demand Forecasting & Food Waste Optimization (Power BI + SQL Project)

## Project Objective
The objective of this project is to analyze restaurant data to improve demand forecasting, reduce food wastage, and enhance profitability. The system helps restaurant owners make data-driven decisions by understanding customer behavior, demand patterns, and operational inefficiencies.

---

## Dataset used
- Orders Dataset  
- Items Dataset  
- Inventory Dataset  
- Waste Log Dataset  
- External Signals Dataset (Weather, Weekend, Festival)  

---

## Questions (KPIs)

- Which items are most frequently ordered?  
- Which items contribute the most to food waste?  
- How does demand vary across weekdays and weekends?  
- What is the impact of festivals on customer demand?  
- How does weather affect order patterns?  
- Which items generate the highest profit?  
- What is the total revenue and profit per restaurant?  
- What is the waste percentage and waste cost?  
- What are the main reasons for food wastage?  
- Which items have high demand but also high waste?  

---

## Process

- Verified data for missing values and inconsistencies  
- Cleaned and formatted datasets  
- Merged multiple datasets using SQL joins  
- Created calculated fields (profit, waste cost, etc.)  
- Performed analysis using SQL queries  
- Built interactive dashboards in Power BI  
- Applied slicers and filters for dynamic analysis

## Project Insight

- A few items contribute to most of the revenue  
- High-demand items also generate high waste  
- Weekends and festivals increase customer demand  
- Weather conditions influence order patterns  
- Expired/spoiled items are the main waste reasons  
- Poor inventory planning leads to higher waste cost  
- Profit varies significantly across items  

---

## Final Conclusion

Using data analytics, restaurants can reduce food waste and improve profitability. This project helps optimize inventory, adjust food preparation, and plan better for peak demand periods. It supports smarter and more sustainable business decisions.
